Engineering Projects

Off-Grid Smart Water Solution

Designed a portable solar-assisted water pumping and purification prototype for off-grid operation with battery-backed and portable backup power support.

Project Overview

This project was developed as a compact off-grid water collection and purification system intended for remote environments where conventional electrical infrastructure may not be available. The system integrates intake tubing, filtration hardware, portable flow components, and a field-deployable operating layout to support practical water collection and treatment. A key engineering feature is its hybrid power concept: the unit can operate using solar-supported input and can also continue running through portable backup sources such as power banks or small external battery devices, improving reliability under varying field conditions.

Key Elements

  • Solar-assisted pumping and filtration concept
  • Portable battery and backup power support
  • Compact modular filtration architecture
  • Off-grid deployment suitability
  • Field testing and practical system integration

IoT Smart Energy Meter

Developed an embedded IoT-based smart metering prototype for real-time electrical parameter monitoring and remote data visualization.

Project Overview

This project demonstrates an IoT-enabled electricity monitoring system built around an ESP32 Wi-Fi microcontroller for signal acquisition, embedded processing, and wireless communication. The system uses an SCT-013 non-invasive AC current sensor and a ZMPT101B AC voltage sensor module to capture electrical measurements required for smart metering. The embedded controller processes the sensor inputs to estimate voltage, current, power, and cumulative energy usage in kWh, then transmits the processed data to the Blynk platform for real-time remote visualization. The project highlights practical integration of sensing, embedded computation, wireless communication, and cloud-connected monitoring in a compact smart energy application.

Key Elements

  • ESP32-based embedded control and communication
  • SCT-013 current sensing
  • ZMPT101B voltage sensing
  • Real-time power and energy estimation
  • IoT dashboard integration using Blynk
  • Remote electrical usage monitoring

Prototype Solar Water Boat

Built a renewable-energy-powered water prototype integrating solar generation, regulated power flow, rechargeable battery backup, and DC motor propulsion.

Project Overview

This project was developed as a small-scale experimental platform to demonstrate how solar energy can be harvested, conditioned, stored, and converted into mechanical motion for water-based mobility. A solar photovoltaic panel acts as the primary energy source, generating electrical power that is passed through a practical regulation stage such as a voltage regulator or DC-DC converter to provide stable electrical output. The regulated power supports a rechargeable battery, which acts as backup storage and can be charged when excess solar power is available. The stored or directly regulated electrical energy is then supplied to a DC motor connected to a propeller-based propulsion mechanism, producing thrust for movement across water. The project reflects practical system integration across renewable power generation, electrical conditioning, energy storage, and electromechanical propulsion.

Key Elements

  • Solar photovoltaic energy harvesting
  • Voltage regulation / DC-DC conversion
  • Rechargeable battery backup and charging
  • DC motor propulsion system
  • Propeller-based thrust generation
  • Renewable-energy mobility integration
